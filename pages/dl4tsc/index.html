Add your model to the platform

If you want to add your model to our platform, you should add an issue to this this github repository. please ensure that:

  • The code of the model should be public, easy to understand and reproducible.
  • The model should be evaluated on the 128 datasets of the UCR Archive.
  • The paper that introduces the model should be published.
