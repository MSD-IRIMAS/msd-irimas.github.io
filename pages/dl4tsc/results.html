Comparing parameters

We plot a trade-off between mean accuracy and number of trainable parameters. The number of trainable parameters is for the feature extraction part without the classification layer. By doing this we show the number of parameters of the network independently of the dataset. On the y-axis, the average accuracy over the 128 datasets of the UCR Archive is presented for each model. The size of the circle representing each model is proportional to the number of parameters of this model.

If you wish to see the performance of all the models on the same plot but for an individual dataset, use the drop down selection here, by default it is set to "All Datasets" to show the average performance on all datasets.
